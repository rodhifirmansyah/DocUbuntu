# Install Ubuntu

Langkah pertama adalah mengaktifkan fitur WSL melalui "Windows Features". Berikut langkahnya:
Buka Command Prompt atau PowerShell dengan hak akses administrator (klik kanan dan pilih "Run as Administrator").
Jalankan perintah berikut untuk mengaktifkan WSL:

wsl --list --online

Update Wsl

wsl --set-default-version 2
wsl --update

# Uninstall Ubuntu

Langkah pertama adalah mengaktifkan fitur WSL melalui "Windows Features". Berikut langkahnya:
Buka Command Prompt atau PowerShell dengan hak akses administrator (klik kanan dan pilih "Run as Administrator").
Jalankan perintah berikut untuk mengUninstall WSL: 

wsl --uninstall

Untuk menghapus distribusi tertentu, gunakan perintah berikut:

wsl --unregister "NamaDristribusi"

Misalnya, jika ingin menghapus Ubuntu, perintahnya:

wsl --unregister Ubuntu

Menghentikan Semua Distribusi Linux Aktif

wsl --shutdown

# Setup Ubuntu

Setup awal Ubuntu

sudo apt update
sudo apt upgrade
